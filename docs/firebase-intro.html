Dans les séances suivantes, nous allons utiliser plusieurs produits de firebase:

Exercices de la semaine

Exercice test_flutter

Vous devez créer un premier projet flutter utilisant Firebase. Ceci inclut les étapes suivantes:

  • Créer mon compte sur firebase
  • Créer un projet dans firebase, appelez le "test-flutter"
  • Créer le projet flutter, appelez le test_flutter (avec un souligné) ça nous permettra de distinguer
  • Connecter les votre projet flutter et votre compte firebase avec flutterfire configure
  • Dans le projet firebase, créez 2 applis
    • test-flutter-ios : projet IOS
    • test-flutter-android : projet Android
  • Suivre et mettre dans les commentaires de votre projet les liens utilisés et les etapes que vous avez réalisées, ça vous sera utile pour vos futurs projets (incluant le TP).
